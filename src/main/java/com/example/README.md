### Примеры кода

* `GenerateRandomNum.java` - генератор рандомных номеров
* `ReverseNumberLoop.java` - реверс вводимых числ при использовании цыкла *`for`*
* `ReverseNumberFor.java` - реверс вводимых числ при использовании цыкла *`while`*
* `CentigradeToFahrenheit.java` и `FahrenheitToCentigrade.java` - конвертация температуры из Цельсия в фарингейты и наоборот
